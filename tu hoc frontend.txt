Tự học frontend

ngắt dòng trong đoạn văn mà vẫn giữ nguyên khoảng cách giữa các dòng.
dùng thẻ <pre>, <br>

thẻ <i> in nghiêng nội dung
thẻ <b> in đậm nội dung
thẻ <em> nhấn mạnh nội dung bên trong (in nghiêng)
thẻ <strong> thể hiện nội dung bên trong là quan trọng (in đậm)

css:
.box1: tất cả element có class box1
.box1, .box2: tất cả element có class box1, tất cả các element có class box2
.box1 .box2: tất cả các element có class box2 nằm trong element có class box1
.box1.box2: tất cả các element có cả 2 class box1 và box2
.box1 > .box2: tất cả các element có class box2 bao quanh trực tiếp bởi element có class box1
